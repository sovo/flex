Flexbox First Look

Additional Resources

Thanks for watching Flexbox First Look! I wanted to share some additional resources on Flexbox that will help you learn more about Flexbox and track support for it.

CSS Flexible Box Layout Module

The best way to learn Flexbox is to dive into the specification itself. It doesn't take long to read, and you'll learn plenty of details that I was unable to go into over the length of this course. Dive in!

Can I Use Flexbox?

Bookmark caniuse.com's page on Flexbox support and keep track of Flexbox and how it's supported across multiple browsers and devices.

CSS Flexbox 2009/2011 syntax mapping

Nice quick reference on the difference between the 2009 and 2011 Flexbox syntax. Keep in mind that the 2011 syntax is considered the "tweener" syntax and is not current.

IE Flexbox reference

Thorough overview of the Flexbox implementation in IE10. Contains all vendor prefixes for -ms- values and explains tweener syntax.

Old Flexbox .vs New Flexbox

Chris Coyier has a nice post here on how to detect the difference between older Flexbox syntax and the newer syntax.

Flexbox - Fast Track to Layout Nirvana?

Fantastic article by Chris Mills that delves into Flexbox, it's history, and the pros and cons of using it.

Why Flexboxes Aren't Good for Page Layout

A leading title to be sure, but Flexbox spec editor Tab Atkins talks about Flexbox, what it's designed to do, and how it will eventually work with other layout modules.
